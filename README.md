# Facial-Recognition

### Set up
```
$ mkdir dataset
$ mkdir trainer
```

### Record new face
```
$ python 01_face_dataset.py
```

### Train model of facial recognition
```
$ python 02_face_training.py
```

### Run facial recognition
```
# without GUI
$ python 03_face_recognition.py

# with GUI
$ python UI.py
```
